PLASMA EDITIONS

Flagship Edition

The Plasma Themed Edition (also known as the Flagship) of Arch Linux GUI helps you install Vanilla Arch Linux with the KDE Plasma Desktop Environment. It is pre-configured and ready to use. It comes with all the neccessary software and settings to help users get started quickly.

Some of the features include:

  • System Wide Dark Theme, including Web Browser
  • Office Suite with MS fonts
  • Printing Services (CUPS)
  • Media Player & Video Conferencing Application
  • Simple Desktop Display Manager

Minimal Edition

The Plasma Minimal Edition of Arch Linux GUI helps you install Vanilla Arch Linux with the KDE Plasma Desktop Environment. It contains very little software. It however comes with certain settings to help users wanting a minimal Vanilla Arch system, get started quickly.

The Plasma Minimal Edition contains very little software. It does not even contain a browser, media player or any other application. The application you get with this edition are:

  • Konsole - Terminal
  • Kate - Advanced Text Editor
  • Dolphin - File Browser/Manager
  • Spectacle - Screenshot Utility
  • Discover - GUI Software Center
  • Firefox - Web Browser

I3 EDITIONS

I3 Edition

The i3 Edition of Arch Linux GUI helps you install Vanilla Arch Linux with the i3 Window Manager.

This edition comes with

  • Gaps enabled by default
  • Kitty - as the terminal emulator. It is transparent by default in this edition.
  • Picom Compositor that enables kitty to have transparency
  • Dmenu - Program Launcher
  • Feh - that handles the background.
  • Paru - An AUR Helper
