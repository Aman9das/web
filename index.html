ALG=Arch Linux GUI

The ALG project, at the moment provides you with 3 editions, featuring the KDE Plasma Desktop Environment. All 3 editions use the same features as mainline Arch Linux, the only difference being in the way of installation. While mainline Arch provides a CLI installation, we at ALG provide a graphical installer, Calamares, to make the process easier.

objective

Pacman

Pacman is Arch Linux's package management system. It is very powerful & it's most important feature is dependency tracking.

Weight

Arch Linux is a lightweight distribution. It is suitable to run on both old as well new hardware. It consumes very less resources.

Rolling

Arch Linux follows a rolling release cycle. It means that the software databases are always up-to-date, with the latest packages.
