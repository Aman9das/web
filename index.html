Features

Easy to Install

Calamares installer makes it easier to install

Stock Desktop Environment

Users wanting to customize their OS from scratch can choose the pure editions which have stock desktop enviroments.

Themed Desktop Environment

Users wanting to get started immediatly can choose the themed editions which have themed desktop enviroments.

Graphical environment

Ready to use graphical environments. Users can start using Arch Linux out of the box, with the themed editions which come with themes, settings and software.

For Intel/AMD computers

Hardware tested on team blue, and team red hardware.

Monthly release, same day as mainline Arch

Monthly rolling release schedule, just like mainline Arch Linux with new ISOs on first of every month.

Legacy/MBR & UEFI/GPT supported

Install in UEFI/GPT or Lgeacy/MBR setup.

Easy to multi-boot

Easily dual-boot/triple boot with Windows and/or other linux distributions.

Encryption using LUKS

All editions come with encryption using LUKS. USers can easily ecrypt drives during installtion.

Printing using CUPS

Themed editions come with CUPS enabled.

Gallery
