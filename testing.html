Studio EDITION

Studio Edition

This edition is geard towards multi-media enthusiasts & producers. It contains the the Low-Latency Xanmod Real-Time Kernel, Pipewire, and many other tweaks to aid in various aspects of multi-media productions, especially the Audio sector.

Software provided here are listed below:

  1. Audio Production - Ardour DAW, Audacity, LMMS, musescore, guitarix, etc
  2. Video Production - Kdenlive, Openshot, qstopmotion
  3. Graphics Production - Blender, GIMP, SynfigStudio, etc
  4. Common Applications: Web Browser, Basic Utilities such as File Broswer, File Archiver, Calculator, Screenshot Utility, etc
